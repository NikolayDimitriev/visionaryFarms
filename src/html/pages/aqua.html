How Air + Aqua
work in harmony.

Air and Aqua were intentionally designed as separate controllers.

This enables easier and more affordable scaling of systems, as users can add exactly the functionality needed. Nothing more, nothing less.

Technical Specification

Here you can check specific details about Aqua. If you’re unclear, feel free to contact our team using the form below.

Technical Data
  • 120-277V Input voltage
  • 50/60Hz Frequency
  • 280х240х45mm Size
  • 0.34kg Weight
  • 0°C to +40°C Optimal ambient temperature
  • 10-90% Relative humidity (non-condensing)
  • 25W Power
  • 8 Output relay channels
  • 500W Max power to each relay channel
  • 4 Channels for concentrate dispensers
  • 2.4 GHz Wi-Fi network operating frequency
  • Fuse inrush current protection
System Functions
  • Wi-Fi network connection
  • Easy setup with the Visionary Farms app
  • Light indicator to display operating mode
Levels
  • 100-2000l Tank volume management
  • 1m-12h Dosage pause
  • 0.1-5.0ms EC level
  • 3.0-10.0pH (2 sensors)
  • 6 Solution control valves
  • 4 Stock solution dispensers
  • 3 Feeding circuits of the nutrient solution
Sensors
  • Ambient temperature sensor
  • Humidity sensor
  • CO2 sensor
  • pH sensor
  • Solution temperature sensor
  • Electrical conductivity sensor
Inputs/outputs
  • Tank fill level sensors RJ45
  • Reserve input
  • Water meter
  • Emergency button: in case of issues, controller goes to idle mode
  • Ejector valves or peristaltic pumps outputs
  • Main Pump (has a separate fuse max power consumption 450W)
  • Fresh water valve
  • Mixing valve
  • Drain valve
  • Outline valve 1
  • Power supply
  • LAN input RJ45
  • EC sensor
  • Nutrient solution temperature sensor
  • Primary pH sensor
  • Secondary pH sensor
  • 4 contactors I2C Bus
  • RS485 Bus
Accuracy
  • 0.5C Temperature measurement error
  • 2% Relative humidity measurement error
  • 50ppm CO2 level measurement error
Warranty
